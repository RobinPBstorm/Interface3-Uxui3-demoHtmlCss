Petit tour d'HTML et CSS avec le groupe UXUI d'interface3

## Deuxiéme partie : HTML

### Pour commencer

Dans un fichier .html, on peut utiliser le raccourcis "!" pour générer la base d'un fichier html.

### Les commentaires

On peut écrire les commentaires comme ceci:

```html
<!-- mon commentaire -->
```

### Les balises

Nous avons rencontré les balises:
 - h1 -> h6 : les titres
 - ul : liste non ordonnée
 - ol : liste ordonnée
 - li : un élément d'une liste
 - p : paragraphe
 - a : ancre
